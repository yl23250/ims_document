* # 功能点

* ## 查询列表
 
   ```
  根据客户名称检索，查询需要查找的客户列表，
  ```

* ## 查看详情
  
  ```
  找到，客户列表表头【客户名称】，点击列内的（蓝色字体）客户名称，如：“华宁县人民医院”，
  即可查看客户资料详情
  ```

* ## 新增

  ```
    点击【添加客户】按钮，根据页面提示，完善客户内容。
      1、基本信息
         建议使用者，先将第一个Tab页，客户[基本信息]
      
    
  ```
